#Erros Codes
**400 - Bad Request**
Dados enviados de forma incorreta ou fora do padrão
**401 - Unauthorized**
Não autorizado
**403 - Forbidden**
Sem autorização suficiente para acessar o recurso desejado
**5xx - Internal**
Erro de Infra-estrutura
