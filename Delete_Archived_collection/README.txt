Once your collections are archived in Team workspace. You can try to delete those APIs using the Postman API, using the below two requests.

Generate API Key, navigating to the below link. More details can be found [here](https://learning.postman.com/docs/developer/intro-api).

- https://go.postman.co/settings/me/api-keys

Also, if you would like to recover the deleted archived collection then:

- Once the collection is deleted, navigate to the [trash](https://go.postman.co/me/trash).
- Now click on "Restore" and select your personal workspace to have these collections back.
