**Introduction**

Cardpayz is a simple, fast, and reliable payment engine with open architecture. It is supported and constantly developing by the community of software engineers with a bold background in payment and e-wallet systems.

Cardpayz Business API - a solution specifically designed for internet businesses in need of multicurrency payment processing. We support all major currencies.


**Production Environment**

There is only a live environment available for integration:
Production environment: https://payments.cardpayz.com


Production always makes real bank transactions, there are no cards from Sandbox available. *You should use real Cards.*
Once you complete integration with the Test user you will be provided with Production credentials.
