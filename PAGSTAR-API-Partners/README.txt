
A *API PAGSTAR PARTNER*  tem o intuito de facilitar a comunicação e integração de sistemas do Partner Pagstar com aplicativos Mobile , operadoras de cartão de crédito e bancos comerciais. 

*Objetivo:*

Receber mensagens para Consultas de Registros na Base do PAGSTAR.
Conectar com Sistemas de pagamento de operadoras de cartão de crédito e bancos, disponibilizar  respostas das conexões com a operadora de cartão de crédito.

**Tecnologia Utilizada:**

Restfull

As operações são feitas por requisições aos endpoints, de acordo com os verbos HTTP, com corpos de mensagem em JSON. As respostas podem ser interpretadas através do código de status HTTP. Isso permite que quaisquer aplicações consumam a API de forma simples e clara, independente da linguagem utilizada.

Para facilitar ainda mais a integração, fornecemos SDKs em todas as principais linguagens de programação. Saiba mais sobre as Rest

https://pt.wikipedia.org/wiki/REST
