# Sobre
Essa API é apenas para usuários devidamente cadastrados e com um plano ativo na ferramenta.

A mesma não é liberada para clientes em teste.

# Endpoint
A URL base da API a ser utilizada sempre será https://whats.agiss.com.br/api/send/TOKEN.

# Formatos e Características
Nossa API aceita apenas o formato JSON, portanto todas as chamadas e requisições usam content type application/json no encode UTF-8.
