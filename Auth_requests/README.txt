# Error Codes
В случае ошибки придет параметр error:
```
{"error":"error_code"}
```
Error codes:
```
401 invalid_login_or_password 
403 user_not_activated
403 user_2fa_forbidden
403 tatt_limit_reached
403 try_is_locked
403 tries_limit_is_exceeded
403 invalid_token
403 invalid_code
403 2fa_required
404 invalid_device
```
