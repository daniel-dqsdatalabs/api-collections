This collection shows how [pre-request scripts][pre-req] in Postman can be used to generate [JSON Web Tokens][jwt] (JWT). This is useful for APIs that need their clients to create JWTs and send them as part of requests.

[pre-req]: https://www.getpostman.com/docs/v6/postman/scripts/pre_request_scripts
[jwt]: https://en.wikipedia.org/wiki/JSON_Web_Token
