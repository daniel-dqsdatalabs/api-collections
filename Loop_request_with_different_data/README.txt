This collection shows how you can loop over the same request while changing the parameters using the Collection Runner and the `postman.setNextRequest()` function.

To try it out, open the collection, then click on "Run" to open the collection runner. Select the query you want to run and run it!
 
This is an answer to the two following questions on the Postman Community Forum:
- https://community.postman.com/t/running-a-request-multiple-times-with-different-data-sets/1064
- https://community.postman.com/t/how-to-loop-through-array-and-use-its-values-in-a-request/6771
