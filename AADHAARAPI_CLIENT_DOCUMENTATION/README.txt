AadhaarAPI provides various validation and verification services via simple REST APIs or web/android gateways. 


*PREPROD URL*	: `https://preprod.aadhaarapi.com`

*PROD URL*	: `https://prod.aadhaarapi.com`

Accessing INIT call and other secure APIs requires `qt_agency_id` and `qt_api_key` in headers. 

- Agency ID can be retirived from my account section on dashboard.
- API KEY can be generated under API key section. 

Note: API key when generated will override any existing API key. Also, do not share this API key with anyone. 

For moving to production share IP addresses to whitelist. Also get in touch with the business team at contact@aadhaarapi.com to get your access.
