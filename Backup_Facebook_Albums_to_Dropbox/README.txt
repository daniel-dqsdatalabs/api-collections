This collection makes use of Facebook's graph API and Dropbox's API to sync all our albums to dropbox. 

## Getting Started

First get Facebook Graph API's access token and Dropbox API access token.

Update the Postman Environment Variables with your token.

Now, run the collection.

## Features

1) Creates Backups of all your facebook albums.

2) Prevents Duplicate Backups.

3) Arranges them into separate folders for every album.
