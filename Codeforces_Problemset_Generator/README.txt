A collection that generates a random [Codeforces](https://codeforces.com) problem set by taking input through [slack slash command](https://slack.com/intl/en-gb/help/articles/201259356-Slash-commands-in-Slack) and pushing the results back to the slack channel. 

>**Slash command syntax:**  
`/getcf cfHandle,minDifficulty,maxDifficulty,problemCount,(optional)tag1;tag2..`  
e.g: `/getcf cooper123,1300,1800,5,math;dp;implementation`
