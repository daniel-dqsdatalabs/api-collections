# Introduction
This API interacts with the store product collections. 
A product collection is used to group selected products for easier browsing on the shop. You can add a name and banner image to a collection. 

# Overview
Through this API you can create, update and list product collections in your store. 
You can also add or remove products from a collection.

# Authentication
By signing in via credentials or social media you can obtain an access token. This access token is used to authenticate the user. 

# Error Codes
Regular HTTP error codes apply. 
* 200 or 204 for successfull requests. 
* 403 for unauthorized requests
* 422 for wrong parameter input. 
* 500 for internal server error.
