# Welcome
This is the API service for the COVID-19 Tracker Canada project. The goal is to provide fast access to the dataset maintained by the COVID-19 Tracker Canada team.


# Get Started
GET [https://api.covid19tracker.ca/summary](https://api.covid19tracker.ca/summary)

See additional routes in the menu.
