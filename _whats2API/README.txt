# W2API - WhatsApp REST API 

Welcome to best REST API for WhatsApp. 
Important to know, it's not an official solution - if you need something professional please don't use this package.

This library provide you an complete solution for WhatsApp REST API without any necessities of code or change anything, obviously, if you want, you can contribute but give this information for us over PR.

### Running
```bash
> npm install && node debug.js
```
