> [Open Data](https://en.wikipedia.org/wiki/Open_data) means the kind of data which is open for anyone and everyone for access, modification, reuse, and sharing.


This collection explores a few examples of endpoints available through the [San Francisco Data](https://datasf.org/opendata/) API, powered by Socrata. Search hundreds of datasets from the City and County of San Francisco. 

![[SF open data website](https://i.imgur.com/lrzZAwO.png)](https://i.imgur.com/lrzZAwO.png)

[Socrata](https://www.tylertech.com/products/socrata) hosts over one hundred different data catalogs for governments, non-profits, and NGOs around the world. Build applications on open data with the Socrata Open Data API (SODA). Read the [Getting Started guide](https://dev.socrata.com/consumers/getting-started.html) or [Socrata's API documentation](https://dev.socrata.com/) to learn more.
