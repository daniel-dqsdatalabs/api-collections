#### Love the collection? Want to build one for yourself? Check this [blog post](https://blog.postman.com/postman-games-public-workspace/)

Let’s have some fun with APIs and test your Postman knowledge. This Postman collection is going to test your Postman knowledge and help you learn more about Postman, the Postman community, and the extended ecosystem. The quiz contains a series of requests that you send to receive questions as a way to test your Postman IQ. If you complete all the levels, you’ll win a prize. 

#### Use the **Visualize** option for each request as a guide to your next steps. 
<br>
