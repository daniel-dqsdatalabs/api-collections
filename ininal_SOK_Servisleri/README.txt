API ininal servislerinin kullanımı için authentication mekanizması olarak oAuth2
kullanılmaktadır ve size iletmiş olduğumuz; ApiKey ve secretkey bilgileri ile Access Token
servisi çağırılarak belirli bir süre geçerli olacak olan accesstoken bilgisi alınır. Aşağıdaki
servisler, bu Access Token servisi ile response içersinde dönen accessToken ile
çağırılmalıdır.
