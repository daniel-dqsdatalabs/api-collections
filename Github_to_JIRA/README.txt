This collection lets you transfer a GitHub issue to JIRA.

You will need the following to make this work:

1. [Personal access token from Github](https://github.com/settings/tokens)
2. [Your basic auth credentials for JIRA](https://developer.atlassian.com/jiradev/jira-apis/jira-rest-apis/jira-rest-api-tutorials/jira-rest-api-example-basic-authentication)
 
[JIRA Documentation](https://docs.atlassian.com/jira/REST/latest/#api/2/issue-createIssue)

[Github Documentation](https://developer.github.com/v3/issues/#get-a-single-issue)
