API agar aplikasi JKN Mobile berkomunikasi dengan Aplikasi SIMARS INNOVA eHOSPITAL. Untuk saat ini terdiri dari:
- Membuat/Melihat Antrian(Janji Poli) Pasien BPJS
- Melihat Booking Operasi

## Header

Pada setiap request terdapat beberapa header yang diperlukan untuk akses ke Aplikasi SIMARS INNOVA eHOSPITAL :
- Content-Type : application/json
- Accept-Type : application/json
- X-Token : --input token hasil dari login--
