# Introduction
This API interacts with the influencer store resource. 

# Overview
This API enabled you to create, update and show an influencer store. 
Listing store invoices is also part of the store API. 

# Authentication
By signing in via credentials or social media you can obtain an access token. This access token is used to authenticate the user. 

# Error Codes
Regular HTTP error codes apply. 
* 200 or 204 for successfull requests. 
* 403 for unauthorized requests
* 422 for wrong parameter input. 
* 500 for internal server error.
