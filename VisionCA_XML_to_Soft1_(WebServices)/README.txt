<b>VisionCA XML to Soft1</b> Application

Σήμερα, όλοι χρησιμοποιούμε το <b>Cloud</b> έστω και άθελα μας. Για παράδειγμα, ανεβάζουμε μια φωτογραφία σε ένα photo website (πχ. το Flickr) και χωρίς να το καταλάβουμε, έχουμε κάνει ήδη το πρώτο μας Cloud Computing. Το </b>Soft1 Cloud ERP</b> αποτελεί μία σύγχρονη λύση μηχανογράφησης που απλοποιεί όλες τις διαδικασίες της επιχείρησής, αποβάλλοντας την πολυπλοκότητα από την καθημερινή λειτουργία. Η εταιρεία μας, πρωτοπόρος στην υιοθέτηση της, βοηθάει κάθε πελάτη στην απόλυτη προσαρμογή στις ιδιαίτερες απαιτήσεις του, με την ελάχιστη επένδυση σε κόστος και πόρους IT. 

Το <b>Soft1 XML</b> application που δημιουργήθηκε από την <b>Vision Computer Applications</b>, χρησιμοποιεί τις τεχνολογίες Cloud και Web Services, που παρέχονται από την Softone και βοηθά στην εισαγωγή ειδών, καθώς και στην ανανέωση των αποθεμάτων τους, από οποιοδήποτε αρχείο XML ενός η περισσοτέρων προμηθευτών σας. 

Με το <b>Soft1 XML</b>, μπορείτε να εισάγετε όλα τα στοιχεία ενός XML αρχείου στο αρχείο ειδών του Soft1, όπως Κωδικός και Περιγραφή, καθώς και Κατηγορίες, φωτογραφίες, αναλυτική περιγραφή και οποιοδήποτε άλλο χαρακτηριστικό είναι διαθέσιμο από την γκάμα των ειδών του ή των προμηθευτών σας. Επίσης σας δίνει τη δυνατότητα να ενημερώνετε είτε on demand είτε είτε αυτόματα, τα αποθέματα που είναι διαθέσιμα στις αποθήκες του προμηθευτή σας.

Επιπλέον, η εφαρμογή <b>Soft1 XML</b> μπορεί να λειτουργήσει και ανάποδα. Δηλαδή μπορείτε να εξάγετε το αρχείο ειδών σας ή όποια από τα είδη της αποθήκης σας επιθυμείτε (περιορισμένα με φίλτρα ευρετηρίου), με τις αντίστοιχες πληροφορίες τους (κωδικό, part number, φωτογραφία κ.τ.λ.) σε αρχείο XML, ώστε αυτό να είναι διαθέσιμο στους πελάτες σας ή σε εξωτερικές εφαρμογές (π.χ. Skroutz, Best Price κ.α.).
