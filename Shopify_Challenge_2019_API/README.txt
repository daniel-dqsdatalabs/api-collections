Welcome to the shopify challenge of 2019!

This was created for the shopify challenge of summer 2019 aimed on creating the barebones of an online marketplace. This api was coded using the Node.js framework Expressjs and uses firebase for a database. 

The full challenge description can be found at https://docs.google.com/document/d/1J49NAOIoWYOumaoQCKopPfudWI_jsQWVKlXmw1f1r-4/edit

#Setup

-You must have node.js installed. then travel to the api's directory and use "npm install" to install the required modules.

To run the web application just do 'node app.js"
