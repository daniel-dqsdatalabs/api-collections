Welcome to the API **marketplace V1**. With it, you can register new custumers, either with suppliers **(PROVIDER)** or beauty salons **(CLIENT)**, register new products by provider, search for product list  and among other services that will be described later. 

Our API  natively supports the **REST** formart.

The system  should make a request  ot the URL https://xpqu8urnj3.execute-api.sa-east-1.amazonaws.com/production
sending the parameters by **GET**, **POST**, **PUT** or **DELETE.** For each request, the response will be a JSON object, which is detailed throughout this documentation.
