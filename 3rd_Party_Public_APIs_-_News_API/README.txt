Get breaking news headlines, and search for articles from over 30,000 news sources and blogs with our news API.

This is a Postman collection for the News API, helping make the News API more accessible and usable by developers. You will need to obtain your own News API key from the API, and add to the environment for your collection before you can use.

- [Website](https://newsapi.org/)
- [Signup](https://newsapi.org/register)
