GoodZap General Purpose API

GoodZap is a WhatsApp messaging and notification solution for your customers.

An account setup interface is now available at https://goodzap-for-all.herokuapp.com/

Only use this API if you want to maintain your own graphical interface within your system.

Authentication token must be requested by email contato.goodzap@gmail.com

Before testing, download the Google Chrome extension at 
https://chrome.google.com/webstore/detail/goodzap-extension-client/hdbkceednledahpbbmonlalimkmaogod?hl=pt-BR&gl=BR  and set your Auth Token on auth settings of installed extension and restart Chrome.
