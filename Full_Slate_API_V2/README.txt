# Steps to execute the CURL

1. Download Desktop version of POSTMAN https://www.postman.com/downloads/

2. Click the "Run in Postman" button from the top right corner of this page

3. Change the URL to your company-key e.g. https://yourcompanykey.fullslate.com/api/v2/...

4. Change the Authorization Token to your API Key or JWT access token

5. Run the CURL examples in POSTMAN
