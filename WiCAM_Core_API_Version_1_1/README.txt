[WiCAM](https://www.wicam.com) develops and sells high-end **CAD / CAM** and **nesting** software for automated 2D and 2.5D CNC manufactuting as well as 3D **unfolding**. 

With the **Core API** [WiCAM](https://www.wicam.com) gives access to to a cloud solution of its main product [PN4000](https://www.wicam.com/en/cadcam-nesting-software/pn4000/).

### &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Main functionality of the Rest API:

* geometry validation
* 2D geometry convertion
* NC machine program conversion
* 3D Assembly splitting
* 3D unfolding
* Calculation based on automated nesting
