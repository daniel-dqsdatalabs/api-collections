Learn how to build and test API requests in Postman.

* Click __Fork__ on the collection overview to copy the collection into your own workspace.
* In your workspace, open the __Postman Classroom__ collection on the left
* Open the first folder `1. Begin training - Requests` > open the first reqeust `0. Start training` and hit __Send__.
* The responses will guide you through the next steps!

*Click __Visualize__ each time you receive a request response.*
