If you’re new to APIs, then you might only be familiar with a few HTTP status codes. To help us learn new HTTP status codes and build our first collection in Postman, let’s check out [HTTP cats](https://http.cat/). 

And if you’re allergic to cats, use [HTTP Status Dogs](https://httpstatusdogs.com/) instead.
